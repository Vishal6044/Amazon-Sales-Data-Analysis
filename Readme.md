# Amazon Sales Data Analysis (EDA) 📊

## Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the **Amazon Sales Data 2025** dataset using **Python**, **Pandas**, **Matplotlib**, and **Seaborn**.

The main objective of this project is to clean, explore, analyze, and visualize the sales data to extract meaningful business insights that can support data-driven decision-making.

---

## Objectives

* Understand the structure of the dataset.
* Identify missing and duplicate values.
* Analyze sales performance.
* Explore customer purchasing behavior.
* Visualize important business metrics.
* Generate actionable insights.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook / VS Code

---

## Dataset Information

The dataset contains Amazon sales transaction records with the following columns:

* Order ID
* Date
* Product
* Category
* Price
* Quantity
* Total Sales
* Customer Name
* Customer Location
* Payment Method
* Status

---

## Exploratory Data Analysis (EDA)

The following analysis was performed:

### Data Exploration

* Loaded the dataset
* Displayed first and last records
* Checked dataset shape
* Displayed column names
* Examined data types
* Generated dataset summary

### Data Cleaning

* Checked missing values
* Checked duplicate records
* Removed duplicate rows (if any)
* Verified data consistency

### Statistical Analysis

* Summary statistics
* Mean
* Median
* Minimum
* Maximum
* Standard Deviation

### Data Visualization

* Category-wise Sales
* Product-wise Sales
* Payment Method Distribution
* Order Status Distribution
* Customer Location Analysis
* Sales Distribution
* Scatter Plot (Quantity vs Total Sales)
* Correlation Heatmap
* Box Plot for Outlier Detection
